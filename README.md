# 🔥 Pyre Shell

Unix shell written in Python.

## Install

Using `uv tool`:

```bash
uv tool install git+https://github.com/mrtotopo/pyre
```

Using `uvx`:

```bash
uvx git+https://github.com/mrtotopo/pyre
```

Using `pipx`:

```bash
pipx install git+https://github.com/mrtotopo/pyre
```

## Usage

After installing, you can run the shell using:

```bash
pyre
```
